# Library Management System (C++)

## 📌 Description
A console-based Library Management System developed in C++ that manages book records efficiently. It allows users to add books, search by ID, issue and return books, and display all records. The system uses file handling to store data persistently.

## 🚀 Features
- Add Book
- Display All Books
- Search Book by ID
- Issue Book
- Return Book

## 🛠️ Technologies Used
- C++
- Object-Oriented Programming (OOP)
- File Handling

## ▶️ How to Run
1. Compile the code:
   g++ library_management_system.cpp -o app
2. Run the program:
   ./app

## 📂 File Used
- books.dat (stores book records)

## 📌 Project Overview
This project simulates basic library operations using a menu-driven approach. It ensures efficient management of books and their availability status using file-based storage.
